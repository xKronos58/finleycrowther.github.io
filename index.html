MY HISTORY

I started coding during the COVID-19 pandemic lockdowns. I started from my deep interest in Minecraft, as i found myself with a lot of free time, i made my own modifications to the game. As i started to delve deeper into coding i started with some basic websites and eventually found myself learning GPU code and creating energy market prediction algorithms.
